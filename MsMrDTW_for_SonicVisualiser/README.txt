[Here are the steps to setup the new alignment in Sonic Visualiser]

[First - Install Sonic Visualiser]
Install Sonic Visualiser from the download page: https://sonicvisualiser.org/download.html
Then run it at least once. This step can be skipped if Sonic Visualiser is already installed.


[Second - Install Miniconda]
In the folder MsMrDTW_for_SonicVisualizer execute "MsMrDTW_for_SonicVisualiser/Miniconda3-py37_4.10.3-MacOSX-x86_64.pkg" and follow the steps.


[Second - Run ./install.sh]
Double click on the file "install.sh"
